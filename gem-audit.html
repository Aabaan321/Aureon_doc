Comprehensive Technical & UI/UX Audit Report

Website Audited gemconstruction.in ↗
Date of Audit March 2026
Prepared By Aureon Studio
Overall Status 🔴 Action Required
Executive Summary: This document serves as a detailed ledger of all structural, functional, and search-engine-related failures currently present on gemconstruction.in. The errors listed below are actively harming user trust, breaking core site functionality, and severely limiting the website's ability to rank on Google.

01 / Critical Errors

Issues currently breaking the website's functionality and visual credibility
🔴 Mass Image Failures (SVG Placeholders)
Impact: 15 out of 17 images on the homepage are failing to load entirely. Instead of displaying high-quality construction portfolios, the website serves raw, broken SVG graphical placeholders. This is likely caused by a broken lazy-loading plugin or caching conflict (e.g., LiteSpeed Cache).
🔴 Broken Primary Navigation Routing
Impact: The button and link mapped to "Commercial Projects" incorrectly redirects users to the residential construction page. This causes extreme confusion and blocks potential high-value commercial clients.
Location of failures:
- Homepage - /services/ - /get-a-quote/

02 / High Priority Issues

Issues severely degrading professional appearance and automated functions
⚠️ Frozen JavaScript Statistics Counters
Impact: The animated number counters designed to dynamically count up to metrics like "Years of Experience" or "Projects Completed" are fundamentally broken. They are frozen at 0+ across almost every page on the site.
⚠️ Unprofessional Internal Email Exposure
Impact: The personal email address nazir.digitals@gmail.com is publicly exposed in the article metadata and authorship tags across the site's blog and news sections instead of an official company email.

03 / Search Engine Optimization (SEO)

Failures preventing the website from ranking on Google and Bing
🔍 Missing Image Metadata (Alt Text)
Impact: 27 images across 8 different pages have completely blank alt attributes. Search engines cannot "see" images; they read the alt text. Without it, the site loses massive amounts of keyword relevance.
🔍 Missing Core Page Titles (H1 Tags)
Impact: The majority of the site's core service pages completely lack an <h1> heading. Search engines use the H1 tag to understand the primary topic of the page. Without it, core pages will not rank properly.
🔍 Conflicting Page Titles
Impact: Conversely, all 3 published blog articles contain multiple <h1> tags on the same page, which confuses web crawlers and dilutes keyword authority.

04 / Medium Priority

Aesthetic improvements and grammatical corrections (Content & Design)
🟡 Auto-Generated Future Blog Dates
Impact: Blog articles and news snippets are stamped with auto-generated placeholder dates located in the future (e.g., "August 16, 2025"). This degrades the authenticity of the company's news feed.
🟡 Grammatical & Capitalization Errors
Impact: The phrase "stylish homes" is improperly capitalized in feature headings across the homepage and service cards, which detracts from the premium feel of a luxury construction firm.
🟡 Header Typo
Impact: The section heading on the homepage reads "Gems Blogs" instead of the grammatically correct "GEM Blogs".

Final Remediation Recommendations

If the decision is made to repair or rebuild gemconstruction.in, the following steps are mandatory:

  • Remove Conflicting Plugins: Disable litespeed-cache and reset the image optimization logic to stop the SVG placeholder loop.
  • Repair DOM Observers: Fix the JavaScript triggering the viewport observer for the 0+ stats counters so the numbers animate correctly on scroll.
  • Correct Routing Architecture: Hardcode the correct absolute path (/commercial-construction/) to all "Commercial" buttons to fix the misdirection to the residential page.
  • Enforce Strict SEO Hierarchies: Ensure every single page has exactly one <h1> tag containing the target local keyword (e.g., "Commercial Construction in Bangalore").
